Discuss the scope of "var", "let", and "const"

In JavaScript var, let and const keywords are used for variable declarations. But they are all not similar in nature. Here is a brief discussion about the scopes of var, let and const:

  • var: var has mainly two scopes. Its either function scoped or global scoped.By being function scoped it means that the variable defined by var is accessible within the function it's defined in. By being global scoped it means that variable defined by var are accessible outside of any code block (such as if statements or loops) in which they are defined.
  • let: Variables declared with "let" are block-scoped, meaning they are limited in scope to the block (curly braces) in which they are declared. This includes loops, conditional statements, and functions.
  • const: Variables declared with "const" are also block scoped meaning they are only accessible within the block in which they are declared.

Tell us the use cases of "null" and "undefined"

In JavaScript "null" and "undefined" are both special values in JavaScript that represent the absence of a value. However, they are used in slightly different contexts and have specific use cases. Following is their use cases:

  • undefined: "undefined" is used for:
    • Checking if a variable is initialized or assigned.
    • Detecting missing function arguments.
    • Identifying missing properties in objects."
  • null: "null" is used for:
    • Explicitly indicating the absence of a value in a variable or object property.
    • Clearing out the value of a variable while still indicating its intentional emptiness.

What do you mean by REST API?

REST API, or Representational State Transfer Application Programming Interface, is an architectural style and set of constraints used for designing web services that allow different software applications to communicate and exchange data over the internet.The API follows certain rules that make it easy for the programs to understand each other. It uses web addresses to identify what data or action is needed, and it uses simple commands like "get," "create," "update," and "delete" to do different things. This helps different programs work together smoothly and share data effectively.
